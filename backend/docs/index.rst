Baseball Stats Dashboard API Documentation
=======================================

Welcome to the Baseball Stats Dashboard API documentation. This API provides endpoints for managing baseball player statistics and generating AI-enhanced player descriptions.

.. image:: _static/api_banner.png
   :alt: Baseball Stats Dashboard API
   :align: center

Overview
--------

The Baseball Stats Dashboard API is built with FastAPI and provides a RESTful interface for managing baseball player data. It includes features for:

* Retrieving player statistics
* Adding new players
* Updating player information
* Generating AI-enhanced player descriptions using OpenAI
* Calculating derived statistics

This API serves as the backend for the Baseball Stats Dashboard web application, demonstrating modern Python development practices and cloud-native architecture.

API Documentation
----------------

The API is fully documented using OpenAPI and can be accessed at the ``/docs`` endpoint when the server is running.
